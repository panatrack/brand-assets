# Panatrack Public Assets

This repository hosts publicly accessible static assets for Panatrack, Inc., 
including logos, icons, and brand images.

## Usage

Files in this repository are intended for use in integrations, email templates, 
and other contexts requiring a publicly accessible URL.

To use a file, navigate to it in the repository, click **Raw**, and copy the URL. 
Raw URLs follow this format:

https://raw.githubusercontent.com/panatrack/brand-assets/main/filename.ext

## Contents

- `logos/` — Primary and variant logo files
- `icons/` — App icons and favicons
